Установка
==========

Для установки приложения запустите скаченный APK файл на своем устройстве.

После установки в главном меню появится ярлык для запуска программы.

Для успешной установки ваше устройство должно обладать следующими минимальными системными требованиями

* Версия Android 4.1 и выше
* Минимальное количество ядер процессора 2
* Оперативная память: 500мб 
* Минимальная частота процессора 1200 МГц
* 5 МБ свободного места на жёстком диске,
