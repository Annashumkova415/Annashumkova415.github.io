Добавление в вспомогательную таблицу¶

В верхнем меню выберете раздел таблицы, в которую хотите добавить данные

Рисунок 1. Главное окно программы.

После нажатия на выбранный раздел откроется таблица с данными содержащимеся в этом разделе

Рисунок 2. Выбор раздела.

Для добавления новой записи нажмите на кнопку «Добавить»

Рисунок 3. Окно выбранного раздела.

Заполните все поля в открывшейся форме и нажмите «ОК» чтобы добавить новое значение в таблицу

Рисунок 4. Форма добавления.

Назад Вернуться в главное меню Далее
